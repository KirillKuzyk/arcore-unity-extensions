Run and debug Google Cloud Anchors API in the Unity Editor
==========================================

## Warning

The official version of the AR Foundation Editor Remote that supports Google Cloud Anchors is not released yet. Please contact me to receive a preview version.


## Description

This package adapts the [ARCore Extensions for AR Foundation](//github.com/google-ar/arcore-unity-extensions) Cloud Anchors to work with the [AR Foundation Editor Remote](//assetstore.unity.com/packages/tools/utilities/ar-foundation-editor-remote-168773) plugin.

Test and debug Google Cloud Anchors right inside the Unity Editor!


## Quick Start

1. Install the AR Foundation Editor Remote plugin.
2. Clone this repo.
3. Add this repo to your project: Package Manager -> Add package from disk... -> Select the 'package.json' file located at the repo's root folder.
4. Make a new build of the AR Companion app to test Cloud Anchors in Editor.
